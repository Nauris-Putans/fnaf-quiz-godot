# Night Shift: Horror Quiz

**Night Shift: Horror Quiz** is a short **2D pixel-art horror quiz game** inspired by classic survival horror, created as a **learning project in Godot**.

The game challenges players to **survive the night by answering questions under pressure**. Every mistake brings you closer to a terrifying end ‚Äî and only those who stay calm until morning will make it out.
